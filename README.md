# üñºÔ∏è Character Image Viewer for SillyTavern

A SillyTavern extension that allows you to view all images related to a character (Description, Author's Notes, Alternate Greetings, etc.) in a sleek and user-friendly gallery.

## ‚ú® Features

* **Deep Scan**: Automatically finds image URLs hidden anywhere in the character's data structure.
* **Floating Gallery**: A draggable and resizable window to browse thumbnails.
* **Immersive Viewer**: 
    * **Frameless Design**: Minimalist interface with floating controls.
    * **Transparent Background**: Perfect for viewing sprites and cutouts over your chat background.
    * **Slideshow Mode**: Navigate through images using on-screen arrows (`<` `>`).
* **Seamless Integration**: Adds a shortcut button directly to the Character Header (next to the delete button) and the Extensions menu.

## üì• Installation

1. Open **SillyTavern**.
2. Navigate to the **Extensions** menu (Puzzle piece icon).
3. Click on **Install Extension**.
4. Paste the URL of this repository:
   `https://github.com/baruvibe/SillyTavern-CharImgViewer`
5. Click **Install**.
6. **Reload** the page (F5).

## üéÆ How to Use

You have three ways to open the gallery:

1.  **Character Panel**: Click the **Image Icon** located next to the "Delete Character" (skull) button in the right panel.
2.  **Extensions Menu**: Open the Extensions drawer (Puzzle icon) and click **"Ouvrir la Galerie"**.
3.  **Chat Command**: Type `/gallery` in the chat bar.
